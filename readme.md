⏰ PWA Alarm – Smart Contract Demo

A simple Progressive Web App (PWA) that demonstrates how a smart contract works using an alarm clock analogy.

How It Works

1. Choose a future time.
2. Click Set Alarm.
3. The app waits.
4. When the selected time is reached, the alarm rings automatically.

If the selected time has already passed, the app displays "Alarm Missed" instead of ringing.

Why This Project?

Smart contracts do not execute immediately. They wait until predefined conditions are met.

This app uses an alarm clock to explain that concept:

- Alarm Time → Predefined condition
- Waiting → Smart contract is deployed but inactive
- Alarm Rings → Conditions are met, so execution happens automatically
- Alarm Missed → The required condition was not met in time

Features

- Simple and beginner-friendly
- Works in any modern web browser
- Installable as a Progressive Web App (PWA)
- Offline support
- Demonstrates automatic execution based on predefined conditions

Technologies

- HTML
- CSS
- JavaScript
- Service Worker
- Web App Manifest

Educational Purpose

This project is designed to help beginners understand the basic idea behind smart contracts through a familiar real-world example.
